# EzLang Compiler

EzLang 是一门以表达式为中心的系统编程语言，目标是提供强类型、泛型、Arena 内存模型、SIMD 向量和现代元编程特性。

## 技术栈

- 词法/语法分析：ANTLR4
- 编译器实现：Python
- 后端生成：llvmlite
- 目标平台：WASI/LLVM

## 开发目标

本项目第一版编译器将覆盖以下内容：

- 词法分析（lexer）和语法分析（parser）
- 语义分析：类型检查、作用域管理、结构体与泛型、函数、控制流
- 中间表示与 LLVM IR 生成
- Arena 内存模型与值语义
- 基础系统接口与 WASI 集成

此外，项目规划不仅限于编译器核心，还包括：

- 内置 lint 和 format 功能，提供代码质量检查与自动格式化
- LSP（Language Server Protocol）支持，提升编辑器体验
- VS Code 插件，直接集成语法高亮、诊断、补全和格式化
